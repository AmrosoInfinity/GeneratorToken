> Saat selesai menambahkan Timezone ulangi perintah */token*

*📍 Menambahkan Timezone*
- Metode request saat ini menggunakan zona waktu untuk menyimpan data request agar dapat membatasi permintaan berlebihan.

*🔘 Menambahkan Kepemilikan Tombol*
- Tombol *Grab Gojek dan Set Timezone* saat command */token* dipanggil adalah milik User. Terdapat batasan waktu 60 detik saat tombol on display. 

*⚠️ Batasan Permintaan*
- Setiap user hanya bisa meminta token **maksimal 3 kali dalam 12 jam**.

*🔒 Aturan Tambahan*
- Jika bot menjadi *admin* di grup:
  - User yang melanggar akan otomatis di-*mute* selama **30 menit**.
  - Permintaan token dari user tersebut akan dihentikan selama **12 jam**. 
- Jika bot *bukan admin*:
  - Bot tetap akan menghentikan respon permintaan token selama **12 jam**.

*📢 Mohon Perhatian*
    Anonymously Admin grup tidak dapat dikenali identitas kepemilikan tombolnya saat AmrosolBot *bukan Admin* grup karena telegram menggunakan virtual user. Pastikan bot adalah Admin grup agar bisa membaca ID user Anonymously yang konsisten sebagai Admin.
